Studio.Namma Clone Website

A responsive and modern clone of the Studio.Namma website built using HTML, CSS, and JavaScript. This project recreates the user interface and core design elements of the original website while focusing on responsive layouts, interactive features, and a seamless user experience.

🚀 Features
Responsive design for desktop, tablet, and mobile devices
Modern and clean user interface
Interactive navigation menu
Smooth scrolling effects
Dynamic JavaScript functionality
Optimized layout and styling
Cross-browser compatibility
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)

🎯 Purpose

The goal of this project is to strengthen frontend development skills by recreating a real-world website design and implementing responsive layouts, styling techniques, and JavaScript-based interactivity.
